 **Superstore Sales Dashboard (Power BI)**

 Project Overview
This project analyzes Superstore sales data to understand revenue, profit, customer segments, and regional performance using Power BI.

 Business Problem
Retail managers need a clear view of sales trends, profit drivers, and regional performance to make data-driven decisions.

 Tools Used
- Power BI
- Excel
- Data Modeling
- DAX

 Key KPIs
- Total Sales: 2M
- Total Profit: 175K
- Total Quantity: 22K
- Average Shipping Days: 4

 Key Insights
- West region contributes the highest sales (~33%)
- Technology category generates the most revenue
- Standard Class shipping dominates order volume
- YoY sales and profit show consistent growth
- 15-day sales forecast helps short-term planning
